What is A2A?

A2A (Agent2Agent Protocol) lets independent agents discover and delegate work to each other. An agent publishes an Agent Card (who it is, where to reach it, what skills it has); a client agent fetches the card and sends a task over JSON-RPC, then receives a stream of lifecycle updates and artifacts back.

The cooperative twist is the input-required state: an agent can pause mid-task and ask the caller for one more detail before it finishes — collaboration, not fire-and-forget.
